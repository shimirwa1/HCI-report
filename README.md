# UNIFIT – Human-Computer Interaction (HCI) Project

## Overview

**UNIFIT** is a Human-Computer Interaction (HCI) project that focuses on designing and evaluating a mobile fitness application for university students. The application aims to encourage healthier lifestyles by helping students overcome common barriers to physical activity such as academic stress, lack of motivation, and limited free time.

This project was completed as part of the **Human-Computer Interaction** course in the Bachelor of Software Engineering program at **Chengdu Neusoft University**.

Rather than focusing solely on interface design, the project follows a complete User-Centered Design (UCD) process—from user research and problem analysis to high-fidelity prototyping and usability evaluation.

---

## Problem Statement

University students often struggle to maintain healthy exercise habits due to:

* Busy academic schedules
* High stress levels
* Limited motivation
* Sedentary lifestyles
* Lack of time for long workout sessions

UNIFIT addresses these challenges by providing quick, guided, and engaging workout experiences designed specifically for student life.

---

## Project Objectives

The project aims to:

* Design a user-friendly mobile fitness application.
* Apply Human-Computer Interaction principles.
* Improve usability through user-centered design.
* Encourage consistent physical activity among students.
* Evaluate interface usability using established HCI methods.

## HCI Principles Applied

This project applies several Human-Computer Interaction principles, including:

* User-Centered Design (UCD)
* Accessibility
* Consistency
* Visibility of System Status
* Recognition Rather Than Recall
* User Control and Freedom
* Error Prevention
* Feedback and Motivation
* Cognitive Load Reduction
* Interaction Design

---

## Usability Evaluation

The prototype was evaluated using multiple usability techniques, including:

* Nielsen's 10 Usability Heuristics
* Cognitive Walkthrough
* Interface Analysis
* User Journey Evaluation
* Accessibility Review

The evaluation identified strengths in navigation, visual consistency, and motivational features while also highlighting opportunities to improve accessibility, progress analytics, and user guidance.
